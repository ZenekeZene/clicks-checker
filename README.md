# Clicks checker

## Installation

```
npm install clicks-checker
```

## Usage

```
import { createClicksChecker } from 'clicks-checker';

createClicksChecker();
```

## Development
To start a development server, run:

```
npm run dev
```


To bundle the library for distribution, run:

```
npm run build
```
